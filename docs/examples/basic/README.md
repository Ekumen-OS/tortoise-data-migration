# Basic example

A very simple project, no structural migrations involved, only data migrations.


## Run the example

```
$ pdm install
$ pdm run python main.py
```

The result should be:

```
manzato@x2:/tmp/test-project$ pdm run python main.py
Available countries:
 - AR | Argentina
 - BR | Brazil
 - US | United States
 - ZA | South Africa
 - GB | United Kingdom
```
